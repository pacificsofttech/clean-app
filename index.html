Introduction

This template is the best suited for business, landing page, mobile app, portfolio, personal websites etc. Mega Business uses the latest HTML5 and CSS3 standards. With the JavaScript library for animation the website looks much interactive and catchy. We have used latest version of Bootstrap 4.2.1. The latest and up-to-date technology have made this template stand among the rest. We have used fonts and icons through CDN which makes it more easy to customize. Changing Heading and Brand Name

You can change the brand name by following steps:
  1. Go to index.html inside MegaBusiness folder of main folder.
  2. Open the file in text editor eg: notepad++.
  3. Change the name of brand from file.

You can change the heading name by following steps:
  1. Go to index.html inside MegaBusiness folder of main folder.
  2. Open the file in text editor eg: notepad++.
  3. Change the name of the heading from file.

Changing Image

Image can be changed in the template as per user choice. You can change the image by two method.

Method 1:

Changing image of html file

  1. Go to index.html inside MegaBusiness folder of main folder.

  3. Change the name of the image from file.

Icons and Fonts

We have used icons and fonts from the font awesome and google fonts respectively. Its more easy to change icons and fonts.

Changing Fonts:
  1. Go to style.css inside css folder which is inside the MegaBusiness folder of main folder.
  2. Check and select the name of the font family.
  3. Add CDN/Link of font family inside head section of index.html
  4. Change font family in style.css inside the css.

Paste CDN/Link inside here by replacing highlighted CDN/Link of prevous font family.

Replace the font family with new one.
